# VICE Debug Tools - Инструкция

## Использование

### 1. Debug Window (Главное окно)
Меню: `VICE > Debug Window`

**Overview** - общая информация (день, час, FPS, количество entities/chunks/NPCs)
**Chunks** - список чанков с поиском, показывает состояние и количество NPC в каждом
**NPCs** - список персонажей с фильтрами (живые, по фракции), поиск по имени/ID, кнопка Inspect для детальной информации

### 2. Scene Visualizer (Визуализация в сцене)
Добавьте компонент `SceneDebugVisualizer` на любой GameObject.
Показывает чанки (цветные границы) и NPC (синие точки) прямо в Scene View.

## Что лучше чем старые инструменты
- Кэширование данных (обновление раз в 0.5 сек вместо каждого кадра)
- Поиск и фильтры для быстрого поиска нужных чанков/NPC
- Цветовая индикация состояний
- Ограничение максимума отрисовки для производительности
- Разделение на маленькие скрипты - легко читать и модифицировать
